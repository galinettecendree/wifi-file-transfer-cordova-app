A propos

Cette application de partage est faite avec Cordova, accompagné des plugins :

  • FileBrowser - Our Code World
  • Cordova Local-Notification Plugin
  • Cordova Splashscreen Plugin
  • Cordova Dialog Plugin

Elle ne collecte aucune donnée personnelle.

Le fichier reste sur le téléphone : l'application le rend juste disponible sur le réseau local en lancant un serveur HTTP qui a son dossier pour racine.

Pour avoir une url raccourcie l'application utilise l'API de Rebrand.ly.
